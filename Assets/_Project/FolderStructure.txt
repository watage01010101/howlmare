Assets/_Project is the root folder for game-specific assets used by this project.

Scenes/
Stores game scenes such as stages, title screens, and test scenes.
Current scene: SampleScene.unity.

Scripts/
Stores C# scripts, grouped by responsibility.

Scripts/Player/
Stores scripts related to player control and player state.
Current main controller: PlayerController.cs.

Scripts/Camera/
Stores scripts related to camera following and camera behavior.
Current controller: CameraController.cs.

Input/
Stores input settings and action definitions.
Current input actions asset: InputSystem_Actions.inputactions.

Physics/
Stores 2D physics materials and other physics-related settings.

Rendering/
Stores rendering and URP-related settings.
Current assets: DefaultVolumeProfile.asset and UniversalRenderPipelineGlobalSettings.asset.

Art/
Intended location for visual assets such as characters, backgrounds, and stage art.

Art/Characters/
Intended location for character images and sprites.
stand_1.png should eventually be moved here.

Art/Stages/
Intended location for stage images and background assets.
pre_stage1.jpg should eventually be moved here.

Settings/
Unity and URP settings are kept in Assets/Settings for now.

_Recovery/
Stores Unity recovery data. Do not edit during normal project work.
